# Fosnie for desktop

## changelog

Added a desktop application for Windows and macOS: pair it with your instance using a code from your profile, then work in it with reliable streaming, system notifications and signed updates you approve before they install.

## site

Fosnie now has a desktop application. Pair it with your instance once and it holds a steady connection: answers stream without stalling, and you are told when one has finished or when an agent is waiting for your approval, even while the window is away.

## detail

The desktop application is the same Fosnie you use in a browser, connected to your own instance: the chats, projects and libraries are the ones already there, and anything you start on the desktop is waiting in the browser. It holds its connection to the instance outside the window, so a long answer survives a minimised window, a closed lid and a change of network instead of stopping mid sentence, and it tells you when an answer is ready or an agent needs your approval. Pairing uses a short code minted from your profile, so the application never asks for your password: its credential belongs to that one computer, lives in the operating system's credential store, and can be withdrawn from the web at any time, which the computer notices within a minute. On its own it reaches nothing on the machine it runs on: the one thing it can do there, working in a folder, happens only in a folder you connect and only with your agreement to each change.
