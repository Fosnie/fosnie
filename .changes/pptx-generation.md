# PowerPoint generation

## changelog

Added PowerPoint (.pptx) generation: editable 16:9 decks with native text, tables, charts and speaker notes.

## site

The platform now builds PowerPoint decks alongside documents, PDFs, pages and spreadsheets. Board packs and findings presentations arrive as real, editable slides rather than something you rebuild by hand.

## detail

A deck used to be the one deliverable the platform could not produce: you got a document, a PDF, a page or a spreadsheet, and rebuilt the slides yourself. Ask for a deck now and you get a genuine PowerPoint file, with text you can retype, tables you can edit cell by cell, charts whose underlying data opens in PowerPoint, and speaker notes under each slide. The assistant chooses what each slide says and which of nine slide layouts carries it, while sizing, colour and positioning are fixed by the platform, so an overlong heading or a list that will not fit is resized or carried onto a follow-on slide instead of running off the page. Every deck is checked before it reaches you: it must reopen cleanly and validate against the published Office file standard, so one that would prompt PowerPoint to repair it never gets delivered.
